# Drum Machine


## Installation and Setup Instructions

You will need [Node](http://nodejs.org/) and [NPM](https://npmjs.org/) installed globally on your machine.

Installation:

`npm install`

To Start Server:

`npm start-craco`

To Visit App:

`localhost:3000`
